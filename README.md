# Sales and Profit Data Visualization using Python

## Project Overview

This project analyzes monthly sales and profit data using Python.
The goal of this analysis is to understand sales trends, compare profit performance across months, and visualize business insights using charts.

## Dataset Information

The dataset contains 12 months of business data with the following columns:

* **Month** – Name of the month
* **Sales** – Total sales amount for that month
* **Profit** – Profit earned from the sales

Example data:

| Month | Sales | Profit |
| ----- | ----- | ------ |
| Jan   | 2000  | 500    |
| Feb   | 2500  | 600    |
| Mar   | 3000  | 800    |
| Apr   | 2800  | 750    |
| May   | 3500  | 900    |

## Tools and Libraries Used

* Python
* Pandas (for data analysis)
* Matplotlib (for data visualization)
* Jupyter Notebook

## Data Analysis Steps

1. Import the dataset using Pandas
2. Explore and understand the dataset
3. Create visualizations to analyze sales and profit trends
4. Identify patterns and business insights

## Visualizations Created

The following charts were created in this project:

* **Line Chart** – Shows monthly sales growth throughout the year
* **Bar Chart** – Compares profit between different months
* **Scatter Plot** – Shows the relationship between sales and profit

## Key Insights

* Sales generally increase during the middle and later months of the year.
* Profit follows a similar trend to sales, showing that higher sales lead to higher profit.
* Data visualization helps businesses easily understand performance trends.
